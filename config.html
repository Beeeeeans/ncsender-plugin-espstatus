Point this at your WLED device's IP. The plugin communicates via the WLED JSON API (POST /json/state) — no custom firmware, no YAML, no compilation. Just flash standard WLED to your ESP32 or ESP8266.

Click Test & Load Effects to verify the connection and fetch the full effects list from your device for use in the Machine States tab.
Set colour, brightness, effect, speed, and intensity for each machine state. Primary is the main colour (col[0]). Secondary is the accent / background colour (col[1]) — used as the unfilled background by the Run progress bar and as the second colour by dual-colour effects. Speed and Intensity (0–255) control animation parameters whose meaning varies by effect.

The Run state uses WLED's built-in Percent effect (fx:98) to fill LEDs proportionally to job completion. Primary = fill colour, Secondary = background colour. Intensity is driven automatically by job progress — no manual control needed.
