All effects must be defined in cnc-status-light.yaml. The bundled YAML includes every effect listed here. Effect names are case-sensitive and must match exactly.
Effect name Strip type Description Recommended for
(none / Solid) Any Steady colour with no animation Idle, Run, Check
Pulse Any Gentle breathe — fades in and out slowly Hold, Door, Tool Change
Fast Pulse Any Rapid flash — conveys urgency Alarm
Strobe Any Hard on/off strobe at fixed interval Alarm, Door
Flicker Any Random candle-like intensity flicker Tool Change, Sleep
Scan Addressable Short bright segment bounces back and forth across the strip Homing
Twinkle Addressable Random pixels sparkle and fade independently Idle, Check
Fireworks Addressable Sparks shoot out from random positions and fade Idle (festive)
Color Wipe Addressable Colour sweeps continuously from one end to the other Homing, Check
Rainbow Addressable Full rainbow wash scrolls across all LEDs (ignores colour setting) Idle (decorative)
Progress Bar Addressable LEDs fill left-to-right proportionally to job completion. Progress % is sent by the plugin to the ESP32 in real time. Run (cutting)
Any strip effects work on both simple single-colour and addressable LEDs.
Addressable effects require per-pixel LEDs (WS2812B, SK6812, APA102, etc.).
Rainbow and Fireworks ignore the colour setting — they use their own colour logic.
